========================
avetime module
========================


The ``postmd.avetime`` module is to help to process the data generated 
from ``fix ave/time`` command in LAMMPS.

We provided ``AveTime`` class to help to read the data from the file generated by ``fix ave/time`` command.
The ``GreenKubo`` class is to help calculate the auto-correlation function.
